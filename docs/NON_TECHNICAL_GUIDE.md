## 網站是什麼？

這個網站是易潔寶公司的線上展示平台，就像一本會自動更新的電子型錄。

**網站包含：**
- 🏢 公司介紹
- 📦 產品展示
- 🏗️ 工程案例（案場）
- 📞 聯絡資訊
- 📍 台灣地圖（顯示服務範圍）

## 開始前的準備

### 需要的工具

1. **電腦**（Windows 或 Mac 都可以）
2. **文字編輯器**（推薦使用 VS Code，免費且簡單）
3. **GitHub Desktop**（用來儲存和發布修改）

### 安裝 VS Code（只需要安裝一次）

1. 前往 https://code.visualstudio.com/
2. 點擊「Download」下載
3. 安裝完成後開啟程式
4. 選擇中文介面（如需要）

### 安裝 GitHub Desktop（只需要安裝一次）

1. 前往 https://desktop.github.com/
2. 下載並安裝
3. 用您的 GitHub 帳號登入

### 取得網站檔案

1. 開啟 GitHub Desktop
2. 點選 File → Clone Repository
3. 輸入網站的位址
4. 選擇儲存位置（建議：桌面）
5. 點擊「Clone」

## 如何新增案場

### 步驟 1：開啟檔案

1. 開啟 VS Code
2. 點選 File → Open Folder
3. 找到網站資料夾（例如：桌面/178mat）
4. 在左側檔案列表中找到並點擊：
   ```
   src → data → projects.ts
   ```

### 步驟 2：複製範本

找到檔案中的任何一個案場（用 `{` 開始，`}` 結束），複製整段。

**範例：**
```typescript
{
  id: "taichung-new-project",           // 案場代碼（英文，不可重複）
  title: "台中新案場",                    // 案場名稱
  location: "台中市西屯區中港路100號",    // 詳細地址
  city: "台中",                          // 城市
  district: "西屯區",                    // 行政區
  year: 2025,                            // 完工年份
  images: [                              // 照片（稍後說明如何上傳）
    "/images/new-project-1.jpg",
    "/images/new-project-2.jpg"
  ],
  description: "這是一個大型住宅社區的案場...",  // 詳細說明
  tags: {
    buildingType: ["住宅大廈"],           // 建築類型
    floorMaterial: ["地磚"],              // 地面材質
    installationType: ["嵌入式"],         // 安裝方式
    framingType: ["AL-620"],              // 框架型號
    surfaceMaterial: ["波羅紋毯面"],      // 面料
    drainageType: ["導水溝"],             // 排水方式
    designFeature: ["風除室-滿鋪"],       // 設計特點
    location: ["大門入口"]                // 位置
  },
  specs: {                                // 規格
    area: "20平方公尺",
    depth: "2cm",
    width: "200cm",
    length: "1000cm"
  }
},
```

### 步驟 3：填入新資料

將複製的內容貼到檔案中，修改成新案場的資訊。

**重要提醒：**
- ✅ 每個案場最後要有逗號 `,`
- ✅ `id` 必須是英文，不可重複
- ✅ 文字要用雙引號 `"` 包起來
- ✅ 數字不用引號（例如：`year: 2025`）

### 步驟 4：儲存檔案

1. 按 `Ctrl + S` (Windows) 或 `Cmd + S` (Mac)
2. VS Code 會自動檢查格式錯誤

### 步驟 5：發布到網站

1. 開啟 GitHub Desktop
2. 左側會顯示您的修改
3. 在左下角「Summary」輸入：`新增XXX案場`
4. 點擊「Commit to main」
5. 點擊上方「Push origin」
6. 等待 3-5 分鐘，網站會自動更新

## 如何修改案場

### 找到要修改的案場

1. 開啟 `projects.ts` 檔案
2. 按 `Ctrl + F` 搜尋案場名稱
3. 找到後直接修改內容
4. 儲存並發布（同新增步驟 4-5）

---

## 如何刪除案場

1. 開啟 `projects.ts` 檔案
2. 找到要刪除的案場（從 `{` 到 `}` 包含逗號）
3. 全部選取並按 Delete
4. 儲存並發布

⚠️ **注意**：刪除前請先備份！

---

## 如何新增產品

### 步驟類似新增案場

1. 開啟檔案：`src → data → products.ts`
2. 複製任一產品的範本
3. 填入新產品資訊
4. 儲存並發布

**產品範本：**
```typescript
{
  id: "new-product",
  name: "新產品名稱",
  category: "收邊框",                    // 分類：面料/收邊框/完整系統
  description: "產品詳細說明...",
  image: "/images/product-new.jpg",
  specifications: {
    material: "鋁合金",
    thickness: "2.0mm",
    width: "可客製化",
    colors: ["銀色", "黑色"],
    features: [
      "特點1",
      "特點2",
      "特點3"
    ]
  },
  applications: ["適用場所1", "適用場所2"]
},
```

## 如何上傳圖片

### 方法 1：直接放入資料夾（建議）

1. 將照片檔案放到：`public/images/` 資料夾
2. 檔案名稱建議用英文（例如：`taichung-project-1.jpg`）
3. 在案場資料中使用：`/images/taichung-project-1.jpg`

### 方法 2：使用圖床（進階）

1. 上傳到 Google Drive 或 Imgur
2. 取得公開連結
3. 直接使用連結

**圖片建議：**
- 📐 尺寸：1920×1080 以上
- 📁 格式：JPG 或 PNG
- 💾 大小：每張不超過 2MB

## 常見問題

### Q1: 我改了內容，但網站沒有更新？

**A:** 請檢查：
1. 是否有儲存檔案？（Ctrl+S）
2. 是否有 Push 到 GitHub？
3. 等待 5 分鐘後重新整理瀏覽器

### Q2: 出現紅色錯誤訊息怎麼辦？

**A:** 通常是格式錯誤，常見原因：
- 少了逗號 `,`
- 少了引號 `"`
- 少了括號 `{` 或 `}`

**快速解決**：用 VS Code 的「復原」功能（Ctrl+Z）回到錯誤前的狀態。

### Q3: 我不小心刪掉重要資料！

**A:**
1. 立即按 `Ctrl + Z` 復原
2. 如果已經儲存，用 GitHub Desktop 的「Discard Changes」
3. 如果已經 Push，聯絡技術人員恢復

### Q4: 如何備份？

**A:**
1. 複製整個 `src/data/` 資料夾
2. 貼到安全的地方（例如：桌面/備份）
3. 建議每月備份一次

### Q5: 可以用手機管理嗎？

**A:** 不建議。雖然可以，但容易出錯。建議使用電腦操作。
