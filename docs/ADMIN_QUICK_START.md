# 🚀 管理後台 - 快速開始

> 5 分鐘學會使用易潔寶管理後台

---

## 第一次使用？跟著做就對了！

### 1️⃣ 啟動管理後台

打開命令提示字元，輸入：

```bash
cd 178mat
npm run dev
```

看到這個畫面就成功了：
```
➜  Local:   http://localhost:3000/178mat/
```

### 2️⃣ 開啟管理後台

在瀏覽器輸入：
```
http://localhost:3000/178mat/admin
```

### 3️⃣ 開始使用

你會看到：
- 📊 **儀表板** - 顯示統計資訊
- 📁 **工程實績** - 管理工程案例
- 📦 **產品管理** - 管理產品型錄

---

## 💡 最常用的功能

### ✨ 新增工程實績（全新！）
```
1. 點擊「新增工程實績」按鈕
2. 填寫工程資訊（標題、地點、城市等）
3. 新增圖片路徑
4. 選擇標籤分類
5. 填寫規格資訊
6. 點擊「新增」儲存
```

### ✏️ 編輯工程實績（全新！）
```
1. 找到要編輯的工程
2. 點擊「編輯」按鈕
3. 修改表單內容
4. 點擊「儲存」
```

### ✨ 新增產品（全新！）
```
1. 點擊「新增產品」按鈕
2. 填寫產品名稱、分類、描述
3. 輸入產品圖片路徑
4. 新增規格、特點、顏色
5. 填寫適用場所
6. 點擊「新增」儲存
```

### 🗑️ 刪除工程/產品
```
1. 找到要刪除的項目
2. 點擊「刪除」按鈕
3. 確認刪除
```

### 💾 備份資料（重要！）
```
1. 點擊頂部「匯出資料」
2. 下載 cms-data.json
3. 保存到安全的地方
```

---

## 🎯 完整流程示範

### 編輯內容並發布

```bash
# 1. 啟動管理後台
npm run dev

# 2. 在瀏覽器編輯內容
# 訪問 http://localhost:3000/178mat/admin

# 3. 匯出資料備份
# 點擊頂部「匯出資料」按鈕

# 4. 停止開發伺服器
# 按 Ctrl + C

# 5. 建置網站
npm run build

# 6. 發布
git add .
git commit -m "更新網站內容"
git push
```

---

## ⚠️ 三件重要的事

1. **定期備份** - 每次編輯前先「匯出資料」
2. **測試再發布** - 確認無誤後再 build
3. **保留舊備份** - 萬一需要還原

## 🆘 遇到問題？

### 無法啟動？
```bash
# 重新安裝依賴
npm install

# 再次嘗試
npm run dev
```

### 資料不見了？
```
1. 點擊頂部「匯入資料」
2. 選擇之前的備份檔
3. 還原資料
```

### 想重新開始？
```
點擊頂部「重置」按鈕
⚠️ 會清除所有修改！
```
