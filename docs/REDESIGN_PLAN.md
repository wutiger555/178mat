# 易潔寶網站重構計劃 - 2025 專業企業設計

## 設計理念

基於 2025 年企業網站設計趨勢，打造具有專業度和設計感的品牌形象網站：

### 核心設計原則

1. **極簡主義與目的性**
   - 清晰的視覺層次
   - 充足的留白空間
   - 精煉的內容呈現
   - 去除模板感的通用元素

2. **強大的品牌系統**
   - 統一的視覺語言
   - 從 LOGO 延伸的配色系統
   - 一致的字體層級
   - 獨特的圖形元素

3. **精緻的動畫設計**
   - "少即是多"的動畫哲學
   - 有目的性的互動反饋
   - 平滑的滾動體驗
   - 視差效果與滾動觸發動畫

4. **影片敘事**
   - 整合 YouTube 官方頻道內容
   - 案例影片展示
   - 施工流程影片

5. **粗體視覺手法**
   - 大膽的排版設計
   - 強烈的視覺對比
   - 高品質的圖片展示

## 技術方案

### 新增依賴套件

```bash
# 高性能動畫庫
npm install gsap

# 平滑滾動
npm install lenis

# React GSAP 整合
npm install @gsap/react

# 進階視覺效果
npm install react-intersection-observer
npm install react-parallax-tilt

# YouTube 數據獲取
npm install youtube-transcript
```

### 動畫技術棧

1. **GSAP** - 複雜時間軸動畫、滾動觸發
2. **Lenis** - 平滑滾動體驗
3. **Framer Motion** - React 組件動畫（保留）
4. **Intersection Observer** - 滾動進入視窗觸發

## 動畫設計規範

### 滾動動畫

```javascript
// 滾動觸發淡入
gsap.from(element, {
  scrollTrigger: {
    trigger: element,
    start: "top 80%",
    toggleActions: "play none none reverse"
  },
  opacity: 0,
  y: 50,
  duration: 1,
  ease: "power3.out"
});

// 視差效果
gsap.to(element, {
  scrollTrigger: {
    trigger: element,
    scrub: true
  },
  y: -100,
  ease: "none"
});
```

### 平滑滾動

```javascript
// Lenis 初始化
const lenis = new Lenis({
  duration: 1.2,
  easing: (t) => Math.min(1, 1.001 - Math.pow(2, -10 * t)),
  smooth: true,
});

// GSAP 整合
lenis.on('scroll', ScrollTrigger.update);
gsap.ticker.add((time) => {
  lenis.raf(time * 1000);
});
```
