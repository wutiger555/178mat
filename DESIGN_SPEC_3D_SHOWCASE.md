# 🏗️ 易潔寶 3D 鋁合金除泥地墊展示頁面 - 設計規格書

## 📋 專案概述

**目標**：創建一個國際水準的 3D 互動展示頁面，向設計師、工程師、建築師展示易潔寶鋁合金除泥地墊的專業應用。

**參考標準**：
- EMCO（德國）- 60+ 年經驗，市場領導者
- Construction Specialties（美國）- Pedimat 系列經典產品
- Forbo Flooring Systems（瑞士）- Coral 系列，50 年經驗

```javascript
// 滑鼠控制

// 觸控控制（移動裝置）
- 單指滑動：旋轉
- 雙指縮放：縮放
- 雙指拖曳：平移

// 熱點交互
- Hover：高亮顯示 + 標註彈出
- Click：展開詳細資訊面板
- 測量工具：點擊兩點顯示距離
```

### 頁面進入動畫
```
1. Logo 淡入（0.5s）
2. 3D 場景加載進度條（1-3s）
3. 場景淡入 + 相機推進動畫（1s）
4. UI 元素依序滑入（stagger 0.1s）
5. 引導提示動畫（第一次訪問）
```

### 場景切換動畫
```
1. 當前場景淡出（0.3s）
2. 相機平滑過渡（0.5s, easeInOut）
3. 新場景淡入（0.3s）
4. 材質數據更新（數字滾動動畫）
```
